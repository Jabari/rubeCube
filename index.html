rübeCube

An interactive, user-configurable Rubik's Cube.

View on GitHub Download .zip Download .tar.gz

A playable Rubik's Cube that lets users change the cube's faces' colors and dimensions.

This app uses HTML5, SASS, & Three.js.

Functions (will) include:

  • Random color
  • Scramble
  • Add & Subtract face dimensions (from 2 x 2 to 13 x 13)
  • and more...
